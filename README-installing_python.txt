usage requirements:

you will need:

python 3.10.11

pillow 10.2.0

if you don't know how to get or use python and pillow conveniently, the easiest solution is:

1: install Thonny

2: in Thonny's menu, go to tools > manage packages > type "pillow" in the search bar > click "search on pypi" > click "Pillow python imaging library (fork)" (should be the first option) > click "install"

you should now be able to run this program from within Thonny
